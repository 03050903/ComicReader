# 漫画阅读器 ComicReader
===========================================================
说明
----------------------------------------------------------------------------------
   大学期间一直喜欢看漫画，很多时候不能在电脑前看，就会下载漫画app在手机上看，那时候就想做个ComicReader App ,可惜自己什么都不会，呵呵。
现在学了android大半年了，就拿漫画app练练手，虽然是个半成品，以后有时间尽量完善了。啊哈哈。。。。数据来自鼠绘漫画的。。。

   这里用到了很多第三方库，当然有些被我改的乱七八糟了，真对不起原作者，啊哈哈希望别打我就可以了
